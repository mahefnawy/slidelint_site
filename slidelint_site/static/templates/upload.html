Slidelint - A slide smell checker. Reads PDF presentation  slides and checks common problems, outputs summary report on the problems.

Presentation slides will be run against the following set of checks:
  • Contrast of text color to background
  • Language tool grammar checker
  • Gendered pronouns
  • Edges danger zones
  • Text font size
